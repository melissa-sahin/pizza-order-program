# 🍕 Pizza Maker

A simple Python project created while learning the basics of Python.

## Description

This program asks the user:
- What food they would like
- The price of the food
- What topping they want

It then displays the completed order using formatted output.

## Skills Used

- Variables
- User input (`input()`)
- Data types (`float`)
- Formatted strings (`f-strings`)
- Output with `print()`

## Example

```text
What food would you like? Pizza
How much does it cost? 12.5
What topping would you like? Mushrooms

Your order is Pizza with Mushrooms. Total price: $12.5
```
